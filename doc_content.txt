📋 SECTION 2: AI MODEL & INFERENCE ENGINE
Prompt for AI Tool:
Build the AI inference pipeline with the following requirements:

MODEL SPECIFICATIONS:
- Architecture: EfficientNetB3 (pretrained on ImageNet, fine-tuned on APTOS 2019)
- Input: 224x224 RGB fundus images
- Output: 5-class classification (Grade 0-4 Diabetic Retinopathy)
- Export format: ONNX for browser inference via ONNX Runtime Web

PREPROCESSING PIPELINE:
1. Resize to 224x224 (maintain aspect ratio, center crop)
2. Normalize with ImageNet stats: mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]
3. Convert to tensor format

GRAD-CAM HEATMAP:
- Generate activation heatmap from final conv layer
- Overlay on original image with red-yellow colormap
- Highlight regions that influenced prediction
- Save as separate image for PDF report

INFERENCE OUTPUT:

- Reject blurry images (blur detection via Laplacian variance < 100)

const CACHE_NAME = 'retinascan-v1';
const urlsToCache = [
  '/',
  '/index.html',
  '/static/css/main.css',
  '/static/js/main.js',

self.addEventListener('install', (event) => {
  event.waitUntil(
    caches.open(CACHE_NAME)
      .then((cache) => cache.addAll(urlsToCache))
  );
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request)
      .then((response) => response || fetch(event.request))
  );
});

const OfflineIndicator = () => {
  const [isOnline, setIsOnline] = useState(navigator.onLine);
  
  useEffect(() => {
    const handleOnline = () => setIsOnline(true);
    const handleOffline = () => setIsOnline(false);
    
    window.addEventListener('online', handleOnline);
    window.addEventListener('offline', handleOffline);
    
    return () => {
      window.removeEventListener('online', handleOnline);
      window.removeEventListener('offline', handleOffline);
    };
  }, []);
  
  return (
    <div className={`badge ${isOnline ? 'bg-green-500' : 'bg-yellow-500'}`}>
      {isOnline ? '✓ Connected' : '⚠️ Offline Mode'}
    </div>
  );
};
